# Niva Academy

Niva Academy is the learning layer for the repo.

The goal is to explain every feature from three angles:

1. What it is.
2. Why Niva needs it.
3. How it fits into this codebase.

The interactive docs site lives in `apps/docs` and currently includes:

- Searchable handbook entries.
- Dark mode.
- Mermaid architecture diagrams.
- An interactive mobile/Firebase/backend/database flow.
- Sprint-oriented learning sections.

Run it locally with:

```bash
pnpm docs:dev
```

The site runs on `http://localhost:3002`.
